# Car Simulator

JavaScript のオブジェクトを用いて，  
車の「状態（スピード・ガソリン）」と「操作（加速・減速・走行）」を管理し，  
DOM 操作によってブラウザ上で操作できるようにしたシンプルなアプリ．

## 概要

- 加速・減速ボタンでスピードを変更
- 走行距離を入力して走行
- スピードとガソリン量をリアルタイムで表示
- ガソリン不足や停止中の場合はエラーメッセージを表示

## 使用技術

- HTML
- CSS
- JavaScript
  - オブジェクト
  - this
  - DOM操作
  - addEventListener

## 今後の拡張案

- スピードに応じた燃費変化
- ゲームオーバー条件の追加
